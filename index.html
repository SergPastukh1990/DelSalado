Two Implementation Paths & Economics

A. eWallet + CryptoHub + Card

  • Launch time: ≈ 2.5 months
  • Included scope: Full B2C + B2B, invoices, multi-asset
  • CapEx range*: 0 – 0 k USD
  • Pros: Fastest go-live, rich functionality
  • Cons: Legacy Py 3.8, extra modules

B. Lean Custom MVP

  • Launch time: ≈ 3–3.5 months
  • Included scope: 8 core features
  • CapEx range*: 0 – 0 k USD
  • Pros: Lowest starting budget, modern stack
  • Cons: B2B & extensions = separate scope

*Final number fixed after Discovery.
